# Used-Car-Price-Prediction
Minor Project - Used Car Price Prediction
📌 Project Details
Used Car Price Prediction
Problem Statement

In the used car market, prices vary widely depending on factors like brand, year, mileage, fuel type, and engine power. It becomes difficult for buyers and sellers to estimate a fair price.

This project aims to build a machine learning model that can predict the price of a used car based on its features.

Objective

To develop a regression model that accurately predicts used car prices using machine learning techniques.

Dataset

The dataset contains information about used cars such as:

Car brand and model
Year of manufacture
Fuel type
Mileage
Engine power
Transmission type
Price (target variable)
Technologies Used
Python
Pandas, NumPy
Scikit-learn
Matplotlib / Seaborn
Jupyter Notebook
Workflow
Data collection and loading
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Feature selection
Model training (Regression model)
Model evaluation
Saving trained model using pickle
Model Used
Linear Regression / (or your model name if different)
Results
The model successfully predicts used car prices with reasonable accuracy.
Performance evaluated using metrics like:
R² Score
Mean Absolute Error (MAE)
Mean Squared Error (MSE)
💾 Output
Trained model saved as .pkl file in the model/ folder
Visualizations stored in the results/ folder
🚀 Conclusion

This project demonstrates how machine learning can be used to estimate used car prices effectively. It helps buyers and sellers make informed decisions in the automobile market.
